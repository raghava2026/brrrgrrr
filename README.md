## ğŸ Getting Started

Since this is a frontend-only project, setting it up is straightforward.

### Prerequisites

Make sure you have **Node.js** installed on your machine.

### Installation

1.  Open your terminal in the project directory.
2.  Install the dependencies:

    ```bash
    npm install
    ```
